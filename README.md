# CleanArchitectureStudy02

### 개요
영화 정보를 제공하는 Open API인 [Movie Database API](https://developers.themoviedb.org)를 활용한 토이 프로젝트입니다.

### 기능
- 인기있는 영화 목록을 리스트 형태로 조회할 수 있습니다.
- 무한 스크롤을 이용해 끊임없이 데이터를 조회할 수 있습니다.

### 프로젝트 구조
- MVVM

### 사용한 라이브러리
- Paging library
- AAC
- Retrofit, Okhttp, Gson
- Coroutines
- Glide
